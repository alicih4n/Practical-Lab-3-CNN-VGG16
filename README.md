# Practical Lab 3 - Vanilla CNN and Fine-Tune VGG16
**Author:** Ali Cihan Ozdemir (Student ID: 9091405)

## Overview
This repository contains a Jupyter Notebook designed to classify a 5,000-image subset of the Dogs vs Cats dataset. The project builds and evaluates two distinct deep learning models for comparison: a custom-designed Vanilla CNN and a Fine-Tuned VGG16 utilizing transfer learning. 

## How to Run
1. Install the necessary dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Open and execute the Jupyter Notebook (`Lab3_Dogs_vs_Cats.ipynb`).

*Note: The notebook is completely self-contained. It automatically downloads the dataset and generates the model weights locally when executed.*
